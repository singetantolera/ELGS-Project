# Ethiopian Law Guidance System - Backend

## 📋 Overview
Backend API for the Ethiopian Law Guidance System, providing legal information access in multiple languages.

## 🚀 Features
- User authentication with JWT
- Multilingual content (English, Amharic, Afaan Oromoo)
- Search functionality with filters
- Bookmark management
- Admin content management
- Email notifications
- Rate limiting
- Security best practices

## 🛠️ Tech Stack
- **Node.js** - Runtime environment
- **Express** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **Bcrypt** - Password hashing
- **Nodemailer** - Email service
- **Express Validator** - Input validation
